# Scripts

Executable audit scripts for the mechanical extension belong here.

First intended scripts:

1. export normalized geometry payloads from existing Tetra constructors;
2. compute contact graphs and candidate hinge axes;
3. evaluate local mobility of selected hinge graphs;
4. run sampled collision checks;
5. export report manifests.

Scripts should write reproducible reports under `results/`, not overwrite source documents.
