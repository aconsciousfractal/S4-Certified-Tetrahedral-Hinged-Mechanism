# Supersession status note

Historical B04/A7c prose that treated signed orientation as sufficient for contact-side closure is superseded.

- A7c alone is exact signed-orientation evidence.
- The B04/A7c bridge is the artefact that adds contact-side and local no-strict-overlap semantics for the six selected hinge rows.
- A7d/the route-wrapper may use the selected rows only with this bridge attached.
